# csharp-calculator-oop

# Опис
.NET 8 консольний застосунок з об’єктно-орієнтованою архітектурою: базова бібліотека калькулятора + розширення у вигляді спеціалізованих калькуляторів (Scientific / Programmer / Statistics) та консольне меню.

# Вимоги
SDK .NET 8 або новіший: https://dotnet.microsoft.com/download
ОС: Windows / Linux / macOS
Бажано: Git 2.4+ для роботи з гілками та PR

# Швидкий старт
dotnet build

# Запустити консольний застосунок
dotnet run --project ./CalculatorApp
